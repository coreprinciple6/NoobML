Supervised Learning

Supervised learning is a subcategory of ML. It is defined by its use of labeled datasets to train algorithms that to classify data or predict outcomes accurately.

Unsupervised Learning

Unsupervised learning uses ML algorithms to analyze and cluster unlabeled datasets. These algorithms discover hidden patterns or data groupings without the need for human intervention.

Classification Problem

A classification problem is when the output variable is a category. A classification model attempts to draw some conclusion from observed values.

Regression Problem

Regression problem finds the correlations between dependent and independent variables. It helps in predicting the continuous variables.
